# Page Reference Ledger

## Purpose

This ledger records page-level evidence before it enters the manuscript. A source may appear in the bibliography or evidence matrix before it appears here, but chapter footnotes should be drafted from this ledger whenever possible.

## Status Legend

- Verified: checked directly in the named source and edition.
- Preview: checked directly in a stable preview, but still needs full-edition confirmation.
- OCR: checked in public-domain OCR, but page images still need confirmation.
- Lead: page or claim identified by a secondary source, but not checked directly.
