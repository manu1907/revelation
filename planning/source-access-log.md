# Source Access Log

## Purpose

This file records attempts to access high-priority sources. It prevents repeated searching and keeps the evidence matrix honest about what has and has not been directly verified.

## Access Status Legend

- Direct: source text checked directly in a legitimate source, publisher preview, public-domain OCR, or authorized page.
- Secondary lead: claim found in a secondary source that names the commentary and page, but the commentary itself still needs checking.
- Bibliographic only: edition/title data checked, but no usable interpretive passage found.
- Not usable: source appears to be pirated, too fragmentary, or not reliable enough for evidence.

## Search Notes

- Avoid citing Scribd or suspicious PDF mirrors in the manuscript.
- Perlego previews are acceptable for planning and may be acceptable for citation only if the passage and page location are stable.
- Public-domain OCR is acceptable for planning, but citation should be checked against page images when possible.
- Secondary sources may identify useful page numbers, but they do not replace checking the named commentary.
